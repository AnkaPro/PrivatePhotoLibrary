Приложение сохраняет выбранные вами фото из фотопленки в библиотеку. Вход защищен паролем, который хранится в Keychain. <br> 
Технологии, которые были использованы на проекте: <br>
UIKit <br>
Autolayout <br>
Keychain <br>
Userdefaults <br>
FileManager <br>
UICollectionView
